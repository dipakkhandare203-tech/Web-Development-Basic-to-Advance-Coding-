Definition List

HTML
HTML (HyperText Markup Language) is the standard markup language used to create and structure content on the World Wide Web. It uses a system of tags and elements to tell web browsers how to display text, images, links, and other multimedia, essentially acting as the "skeleton" or blueprint of a webpage

CSS
CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation and visual formatting of a document written in a markup language like HTML. It controls the look and feel of a webpage, including colors, fonts, layouts, and positioning.

JAVASCRIPT
JavaScript is a versatile, high-level programming language for the Web that enables the creation of dynamic and interactive experiences on websites. It works alongside HTML (structure) and CSS (styling) to form the three core technologies of web development.
